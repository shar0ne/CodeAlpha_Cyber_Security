# 🛡️ CodeAlpha Cyber Security Internship - Project Hub

Ce dépôt rassemble l'ensemble des travaux et projets pratiques réalisés dans le cadre du stage en cybersécurité chez **CodeAlpha**. L'objectif est de développer des compétences concrètes allant de l'analyse réseau à l'audit de code sécurisé et à la détection d'intrusions[cite: 2].

---

## 📂 Structure du Dépôt & Navigation

L'arborescence est segmentée par tâche pour une documentation claire, modulaire et structurée :

*   📂 **[TASK1](./TASK1/README.md) : Basic Network Sniffer** — Script Python de capture et d'analyse de paquets réseau[cite: 2].
*   📂 **[TASK2](./TASK2/README.md) : Phishing Awareness Training** — Module web interactif de sensibilisation au phishing[cite: 2].
*   📂 **[TASK3](./TASK3/README.md) : Secure Coding Review** — Audit de sécurité OWASP Top 10 sur une application Flask[cite: 1, 2].
*   📂 **[TASK4](./TASK4/README.md) : Network Intrusion Detection System** — Déploiement de Snort et rapports de détection d'intrusions[cite: 2].

---

## 📱 Aperçu des Tâches Réalisées

### [TASK 1: Basic Network Sniffer](./TASK1/)
*   **Description :** Outil en Python utilisant `scapy` pour capturer le trafic réseau en temps réel, décoder les entêtes des protocoles et afficher les adresses IP sources/destinations ainsi que les payloads[cite: 2].

### [TASK 2: Phishing Awareness Training](./TASK2/)
*   **Description :** Plateforme web d'apprentissage pédagogique (HTML/CSS/JS) vulgarisant les techniques d'ingénierie sociale, incluant des exemples d'identification de sites de phishing et un quiz d'évaluation interactif[cite: 2].

### [TASK 3: Secure Coding Review](./TASK3/)
*   **Description :** Audit statique et manuel d'une application Flask vulnérable, rédaction d'un rapport complet détaillant 10 vulnérabilités critiques (CWE, PoC) et fourniture d'une version corrigée et remédiée[cite: 1, 2].

### [TASK 4: Network Intrusion Detection System](./TASK4/)
*   **Description :** Configuration et déploiement d'un IDS Snort avec des règles personnalisées pour identifier des activités suspectes et extraction de graphiques analytiques (sévérité, chronologie, types d'alertes)[cite: 2].
